# Healthcare AI Project 🏥🤖

## 📌 Overview
This project is an AI-based healthcare system that predicts possible diseases based on patient symptoms and suggests treatments.

## 🚀 Features
- Disease Prediction using ML
- Symptom Analysis (BERT - simulated)
- Treatment Suggestion (LLM - simulated)
- REST API using Django
- Frontend using React

## 🛠️ Tech Stack
- Backend: Django REST Framework
- Frontend: React.js
- ML Model: Custom logic
- Database: SQLite

## ⚙️ How to Run

### Backend
cd backend
python manage.py runserver

### Frontend
cd frontend
npm install
npm start

## 🔗 API Endpoint
POST: http://127.0.0.1:8000/create/
